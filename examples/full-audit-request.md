# Example Request

```text
$computer-doctor

Do a full audit of this computer and its software environment.
Inspect system configuration, startup behavior, security posture, performance, storage pressure, installed apps, and developer tooling.
Do not make changes unless I approve them.
```

## Good Follow-Ups

- "Prioritize only the top 10 highest-impact issues."
- "Focus more on security and background processes."
- "Turn approved safe fixes into a shell script, but do not run it yet."
