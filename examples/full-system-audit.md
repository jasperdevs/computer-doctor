# Example Prompt

```text
$computer-doctor

Perform a full audit of this computer as a working environment, not just a settings review.

Inspect system configuration, startup behavior, persistent background activity, security posture, suspicious processes or persistence, performance bottlenecks, storage hygiene, installed software, developer tooling, and quality-of-life improvements.

Rules:
- default to read-only auditing
- do not delete, uninstall, disable, or clean anything without explicit approval
- prefer evidence over guesses
- focus on high-impact findings
- tailor recommendations to how this machine appears to be used
- include open-source replacements when genuinely useful
```

## Example Output Shape

1. Executive summary
2. Security findings
3. Performance findings
4. System and internal configuration findings
5. Startup and background findings
6. Software and app recommendations
7. Developer environment findings
8. Prioritized action plan
9. Limitations
